# 平台地牢系统 - 自由组合与程序化生成

**核心理念**: 你建造平台组合，游戏生成内部地牢
**双层随机**: 平台布局(玩家) + 内部结构(程序化)
**独特卖点**: 每个平台内部都是独特的探索空间

---

## 🎮 核心概念

### 宏观层: 平台组合 (玩家控制)

**玩家做什么**:
```
选择平台位置 → 连接平台 → 形成基地布局
    ↓
决定平台类型 → 决定内部主题 → 决定功能
    ↓
策略性布局 → 创造独特基地
```

### 微观层: 地牢生成 (程序化自动)

**游戏做什么**:
```
平台建成 → 触发内部生成
    ↓
根据平台类型 → 生成对应主题地牢
    ↓
随机房间布局 + 随机道具 + 随机敌人
    ↓
每个平台都独一无二
```

## 🏗️ 平台类型决定内部主题

### R&D平台内部 - "实验室迷宫"

**生成规则**:
```
房间数量: 5-8个随机房间
房间类型:
├─ 研究室 (研究设备，数据终端)
├─ 储藏室 (材料箱，研究样本)
├─ 测试区 (实验装置，可能有危险)
├─ 走廊 (连接房间，可能有巡逻机器人)
└─ 核心室 (特殊研究项目，稀有奖励)

敌人:
├─ 实验体 (泄漏的生物样本)
├─ 保安机器人 (失控的防御系统)
└─ 疯狂研究员 (实验事故)

道具:
├─ 研究笔记 (提供加成)
├─ 稀有材料 (高品质材料)
└─ 蓝图碎片 (解锁新科技)
```

**视觉风格**:
- 白色/蓝色调 (科技感)
- 玻璃墙，全息屏幕
- 实验设备和管线

### Combat平台内部 - "武器库要塞"

**生成规则**:
```

房间类型:

├─ 战术装备 (防御装备)
└─ 战斗手册 (战斗经验)
```

**视觉风格**:
- 军绿色/灰色调

### Support平台内部 - "补给站网络"

**生成规则**:
```

└─ 饥饿的老鼠 (小型敌人)

道具:
├─ 大量材料 (Materials, Fuel)
├─ 补给箱 (恢复道具)
```

**视觉风格**:
- 橙色/黄色调 (工业感)
- 货架，箱子，传送带
- 明亮照明，整洁有序

### Intel平台内部 - "情报中心迷宫"

## 🎲 程序化生成系统

### 生成算法

**步骤1: 房间生成**
```gdscript
func generate_platform_interior(platform_type: String):
    var room_count = get_room_count(platform_type)  # 4-12随机
    var rooms = []

    for i in range(room_count):
        var room = generate_room(platform_type)
        rooms.append(room)

    return connect_rooms(rooms)  # 连接房间
```

**步骤2: 房间连接**
```gdscript
func connect_rooms(rooms: Array):
    # 使用最小生成树算法
```

## 🎯 为什么这样设计好玩?

### 1. 双层随机性

**宏观层 (玩家控制)**:
- 平台位置选择

- 基地整体布局

**微观层 (程序化生成)**:

**未知感**:

**内部战斗策略**:
- 利用地形优势

### 4. 重玩价值

**每个基地都不同**:

**每次远征都不同**:

## 💡 技术实现细节

### 数据结构

**平台内部数据**:
```gdscript
class PlatformInterior:
    var seed: int                    # 随机种子
    var platform_type: String        # 平台类型
    var rooms: Array[Room]          # 房间列表
    var corridors: Array[Corridor]  # 走廊列表
    var enemies: Array[Enemy]       # 敌人列表
    var items: Array[Item]          # 道具列表
    var is_explored: bool = false   # 是否已清理

class Room:
    var room_type: String           # 房间类型
    var position: Vector2           # 在网格中的位置
    var size: Vector2              # 房间大小
    var doors: Array[Door]         # 门连接
    var items: Array[Item]         # 房间内道具
    var enemies: Array[Enemy]      # 房间内敌人
    var is_explored: bool = false  # 是否已探索
```

**生成伪代码**:
```gdscript
func generate_interior(seed: int, platform_type: String) -> PlatformInterior:
    var rng = RandomNumberGenerator.new()
    rng.seed = seed

    var interior = PlatformInterior.new()
    interior.seed = seed
    interior.platform_type = platform_type

    # 生成房间
    var room_count = rng.randi_range(4, 12)
    for i in range(room_count):
        var room = generate_room(rng, platform_type)
        interior.rooms.append(room)

    # 连接房间
    interior.corridors = connect_rooms(interior.rooms, rng)

    # 填充道具和敌人
    populate_interior(interior, rng)

    return interior
```
