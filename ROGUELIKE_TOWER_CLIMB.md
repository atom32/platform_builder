# Roguelike爬塔玩法设计

**核心理念**: 将Roguelike元素融入海上基地建造
**灵感**: 权游类Roguelike + 传统爬塔
**独特卖点**: 每次游戏都不同的海上冒险

---

## 🎮 核心玩法：深海探索

### 游戏循环

```
基地建造 (大本营)
    ↓
组建远征队 (选择员工)
    ↓
探索深海 (Roguelike爬塔)
    ├─ 层1: 浅海 (简单)
    ├─ 层2: 中海 (中等)
    ├─ 层3: 深海 (困难)
    └─ 层4: 深渊 (极难)
    ↓
获得奖励 (稀有资源/员工/模块)
    ↓
返回基地 → 建造更多平台 → 探索更深
```

---

## 🏗️ 双系统设计

### 系统1: 母港基地 (大本营)

**特点**:
- ✅ 永久存档
- ✅ 逐渐成长
- ✅ 建造平台
- ✅ 招募员工

**作用**:
- 提供资源支持远征
- 研发新技术
- 存放探险获得的奖励

---

### 系统2: 深海探险 (Roguelike)

**特点**:
- ⚠️ 临时存档
- ⚠️ 死亡重来
- ⚠️ 随机生成
- ⚠️ 层层深入

**核心循环**:
```
选择远征队
    ↓
进入深海 (5层塔)
    ├─ 每层: 随机事件/战斗
    ├─ 途中: 收集资源/遇到敌人
    ├─ 决策: 继续前进? 撤退?
    └─ 结果: 死亡=失去一切, 撤退=保留部分
    ↓
返回基地 (携带奖励)
```

## 🎯 Roguelike核心元素

### 1. 随机性 (Randomness)

#### 平台布局随机化
```
每次远征:
├─ 深海地图随机生成
├─ 资源分布随机
├─ 敌人配置随机
└─ 宝箱位置随机
```

**实现**:
```gdscript
// 使用已有的程序化生成系统
深海地图生成器:
├─ 每层5-10个平台
├─ 随机模块组合
├─ 随机资源分布
└─ 随机敌人配置
```

#### 员工技能随机化
```
每个员工:
├─ 基础技能: 1-5星
├─ 特殊能力: 随机抽取
└─ 性格特征: 随机生成
```

**特殊能力**:
```
Combat专家:
├─ "致命一击" (15%概率双倍伤害)
├─ "防御专家" (减少30%伤害)
└─ "狂战士" (攻击力×2, 防御×0.5)

Research专家:
├─ "快速学习" (研究×2)
├─ "幸运发现" (20%概率额外奖励)
└─ "完美主义" (100%成功率)
```

---

### 2. 永久死亡 (Permadeath)

#### 远征失败 = 损失

**当前**: 远征失败只损失奖励
**改进**: 远征死亡 = 员工永久牺牲

```
远征死亡条件:
├─ 血量归零
├─ 弹药用尽
├─ 时间耗尽
└─ 特殊事件 (海怪吞噬)

损失:
├─ 员工永久死亡 ❌
├─ 装备全部丢失
└─ 探险进度丢失

保留:
├─ 基地平台 ✓
├─ 其他员工 ✓
└─ 已解锁技术 ✓
```

**为什么好**:
- ✅ 死亡有代价
- ✅ 每次都紧张
- ✅ 需要策略决策

---

### 3. 过程生成 (Procedural Generation)

#### 深海地图生成

```
每层结构:
├─ 入口平台 (安全区)
├─ 3-5个战斗平台
├─ 1-2个资源平台
├─ 1个Boss平台 (每层)
└─ 出口 (下一层或返回)

地形生成:
├─ 平台位置: 随机分布
├─ 连接方式: 随机链路
├─ 视觉外观: 程序化模块
└─ 敌人配置: 难度递增
```

**实现**:
```gdscript
// 深海地图生成器
class_name DeepSeaGenerator

func generate_layer(layer_depth: int) -> Dictionary:
    var map = {
        "platforms": [],
        "connections": [],
        "enemies": [],
        "resources": {}
    }

    # 根据深度生成
    match layer_depth:
        1:
            map["enemies"] = generate_weak_enemies()
            map["resources"] = {"common": 10, "rare": 2}
        2:
            map["endmies"] = generate_medium_enemies()
            map["resources"] = {"common": 5, "rare": 5, "epic": 1}
        3:
            map["enemies"] = generate_strong_enemies()
            map["resources"] = {"rare": 8, "epic": 2}
        4:
            map["enemies"] = generate_boss()
            map["resources"] = {"epic": 5, "legendary": 1}

    return map
```

### 4. 爬塔进度 (Tower Climb)

#### 5层深海结构

```
Layer 1: 浅海 (水深20m)
├─ 敌人: 弱海怪
├─ 奖励: 普通资源
├─ 风险: 低
└─ 时间: 5-10分钟

Layer 2: 中海 (水深50m)
├─ 敌人: 中等海怪
├─ 奖励: 稀有资源
├─ 风险: 中
└─ 时间: 10-15分钟

Layer 3: 深海 (水深100m)
├─ 敌人: 强海怪
├─ 奖励: 史诗资源
├─ 风险: 高
└─ 时间: 15-20分钟

Layer 4: 深渊 (水深200m)
├─ 敌人: Boss级海怪
├─ 奖励: 传说资源 + 稀有员工
├─ 风险: 极高
└─ 时间: 20-30分钟

Layer 5: 海底墓场 (秘密区域)
├─ 敌人: 古神级Boss
├─ 奖励: 独特模块 + 特殊能力
├─ 风险: 死亡率极高
└─ 时间: 30-60分钟
```

---

### 5. 解锁与成长 (Progression)

#### 永久进度

```
基地建设:
├─ 更多平台 → 更强战斗力
├─ 研发技术 → 远征加成
└─ 员工培养 → 特殊能力

每次远征后:
├─ 保留的经验值
├─ 解锁的新内容
└─ 发现的新图纸
```

#### 解锁树

```
深海探索 → 解锁新平台类型:
├─ Layer 1 解锁: "深海R&D平台" (产量×1.5)
├─ Layer 2 解锁: "水下战斗平台" (水战专用)
├─ Layer 3 解锁: "资源采集平台" (稀有资源)
├─ Layer 4 解锁: "研究站" (自动研究)
└─ Layer 5 解锁: "海底都市" (终极目标)
```

## 🎮 详细玩法设计

### 远征准备阶段

```
1. 选择远征队
   ├─ 3-5个员工
   ├─ 每个员工有技能
   └─ 需要平衡配置 (Combat/Support/Medical)

2. 装备远征队
   ├─ 消耗资源 (Materials/Fuel)
   ├─ 准备补给品 (医疗包/弹药)
   └─ 选择装备 (影响战斗)

3. 出发
   └─ 进入深海 (动画效果)
```

---

### 探索阶段 (核心玩法)

#### 战斗系统

```
遭遇敌人:
├─ 显示敌人形象 (海怪/海盗/遗迹守卫)
├─ 显示血条 [███████░] 90%
├─ 显示战斗力对比 [你: 5 vs 敌: 3]
└─ 选择: 战斗/规避/撤退

战斗过程:
├─ 自动战斗 (数值对抗)
├─ 实时血量变化动画
├─ 技能触发 (员工特殊能力)
└─ 结果: 胜利/失败/撤退
```

#### 资源采集

```
发现资源平台:
├─ 采集材料
├─ 发现宝箱
├─ 古代遗物
└─ 稀有员工 (极低概率)

采集风险:
├─ 可能触发陷阱
├─ 可能唤醒守卫
└─ 需要决策: 继续还是撤退?
```

#### 决策点

```
每个平台:
├─ 继续深入 (风险↑, 奖励↑)
├─ 撤退返回 (保留50%战利品)
└─ 深入探索 (额外分支, 风险↑↑)
```

---

### 返回阶段

```
成功返回:
├─ 携带奖励
├─ 员工获得经验值
└─ 解锁新内容

死亡:
├─ 员工永久失去
├─ 装备全部丢失
└─ 需要重新招募
```

```

├─ 选择远征队
```

## 💡 为什么这个设计好?

**树形平台** → **探索树形地图**
```
基地: HQ → 子平台 → 孙平台
探险: Layer 1 → Layer 2 → Layer 3
```

**完美契合！**

### 2. 利用现有系统

**远征系统** → **升级为完整探险**
**员工系统** → **作为探险队员**
**资源系统** → **探险消耗**
**模块系统** → **探险奖励**

**所有系统都能用上！**

### 3. 增加重玩性

**每次都不同**:
- 地图随机
- 员工随机
- 奖励随机

**长期目标**:
- 解锁所有Layer
- 收集所有遗物
- 培养完美团队
