# Assets Folder

This folder contains all game assets (art, audio, etc.).

## Folder Structure

- **textures/** - 2D images for UI and materials
- **models/** - 3D models for platforms and objects
- **audio/sfx/** - Sound effects
- **audio/music/** - Background music
- **fonts/** - Custom fonts
- **shaders/** - Custom visual shaders

## Current Status

**Prototype Phase** - Using procedural generation and placeholder geometry.
