# Game Configuration

This folder contains game configuration files for balancing and settings.

## Configuration Files

### game_config.tres
Main game configuration including:
- Starting resources
- Platform limits
- Game balance parameters

### balance_config.json
Balance parameters for easy tweaking:
- Platform production rates
- Build costs
- Expedition rewards
- Combo bonuses

## Current Status

Configuration is currently hardcoded in `platform_data.gd`. Future versions will move this data to these configuration files.
