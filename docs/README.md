## Design Philosophy

### Core Principles

1. **Tree-Based Expansion**
   - Platforms expand in a tree structure (not grid-based)
   - Each platform can have up to 6 child platforms
   - Unlimited growth potential

2. **Strategic Depth**
   - Combo system rewards intelligent layout planning
   - Different platforms provide different bonuses
   - Expeditions require strategic force composition

3. **Accessibility**
   - Simple controls
   - Clear visual feedback
   - Easy to understand, hard to master

### Game Loop

1. **Build** → Construct platforms to expand base
2. **Produce** → Platforms generate resources over time
3. **Optimize** → Create combos for bonuses
4. **Expedition** → Launch missions for more resources
5. **Expand** → Repeat and grow
