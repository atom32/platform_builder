# 雇佣兵海上基地 - 战斗地图设计

**核心理念**: 你建设的基地就是你的战场
**独特卖点**: 每个玩家的基地布局 = 独特的战斗地图
**避免复杂**: 无装备系统，专注于平台布局和员工策略

---

## 🎮 游戏主题定位

### 放弃的是什么

**不再使用**:
- ❌ "合金装备"品牌 (避免法律问题)
- ❌ 复杂的装备/武器系统
- ❌ 角色等级系统
- ❌ 复杂的技能树

### 保留的核心

**核心体验**:
- ✅ 海上平台基地建设
- ✅ 雇佣兵管理
- ✅ 基地作为防御阵地
- ✅ 远征探险 (Roguelike)
- ✅ 策略布局

---

## 🏰 双层游戏结构

### 第1层: 战略层 (基地建设)

**玩家是基地指挥官**
```
建造平台 → 扩展基地 → 布局防御
    ↓
雇佣兵部署 → 平台分配 → 战力配置
    ↓
资源生产 → 支持远征 → 持续发展
```

**核心玩法**:
1. **平台布局策略**
   - Combat平台放在哪里最有效?
   - Support平台如何支援前线?
   - R&D平台是否需要保护?

2. **防御工事设计**
   - 平台连接方式 = 防御路径
   - 树形结构 = 多层防线
   - 部门组合 = 战术加成

### 第2层: 战术层 (战斗/远征)

**两种战斗模式**:

#### A. 基地防御战 (PvE)
```
敌人进攻你的基地
    ↓
在你建造的平台上战斗
    ↓
利用你的平台布局优势
    ↓
击退攻击 → 获得战利品
```

#### B. 深海远征 (Roguelike)
```
组织雇佣兵小队
    ↓
探索深海 (5层塔结构)
    ↓
随机战斗/资源/事件
    ↓
返回基地或全军覆没
```

## ⚔️ 雇佣兵系统 (无装备设计)

### 雇佣兵属性 (简化但有意义)

**核心属性**:
```
雇佣兵
├─ 姓名: 随机生成
├─ 专业: Combat/Support/Medical/Intel/Engineering
├─ 技能等级: 1-5星
├─ 特殊能力: 每个专业1个
└─ 状态: 健康/受伤/死亡
```

**为什么不需要装备?**
- ✅ 平台本身提供武器/防御
- ✅ 专业决定战斗方式
- ✅ 技能等级决定威力
- ✅ 特殊能力提供策略深度

### 专业能力

**Combat雇佣兵**:
```
战斗方式:
├─ 自动攻击 (近战/远程)
├─ 伤害 = 技能等级 × 平台加成
└─ 特殊能力: "致命一击" (15%概率双倍伤害)

平台协同:
└─ Combat平台: 伤害×2
```

**Support雇佣兵**:
```
战斗方式:
├─ 修复受损平台
├─ 补给队友弹药
└─ 特殊能力: "紧急增援" (召唤临时平台)

平台协同:
└─ Support平台: 修复速度×2
```

**Medical雇佣兵**:
```
战斗方式:
├─ 治疗受伤队友
├─ 复活倒地的雇佣兵
└─ 特殊能力: "群体治疗" (所有队友恢复HP)

平台协同:
└─ Medical平台: 治疗效果×2
```

**Intel雇佣兵**:
```
战斗方式:
├─ 标记敌人弱点
├─ 预警敌人进攻
└─ 特殊能力: "战术分析" (显示敌人攻击模式)

平台协同:
└─ Intel平台: 预警时间×2
```

**Engineering雇佣兵**:
```
战斗方式:
├─ 强化平台防御
├─ 改造平台武器
└─ 特殊能力: "临时工事" (短时间内防御×3)

平台协同:
└─ R&D平台: 强化效果×2
```

## 🗺️ 基地作为战斗地图

### 核心概念: 平台 = 战场区域

**每个平台 = 一个战斗区域**
```
R&D平台
├─ 区域特性: 提供"研究强化"buff
├─ 部署效果: Engineering雇佣兵在这里→强化效果×2
├─ 战略价值: 后方支援阵地
└─ 视觉: 实验室设备，作为治疗/强化点

Combat平台
├─ 区域特性: 自动炮塔
├─ 部署效果: Combat雇佣兵这里→伤害×2
├─ 战略价值: 前线火力点
└─ 视觉: 武器系统，作为防御据点

Support平台
├─ 区域特性: 补给站
├─ 部署效果: Support雇佣兵→修复×2
├─ 战略价值: 中继补给点
└─ 视觉: 补给箱，作为恢复点
```

### 战斗流程 (基地防御战)

```
阶段1: 警报
├─ Intel平台提前预警 (30秒)
├─ 显示敌人进攻路线
└─ 玩家部署雇佣兵到平台

阶段2: 敌人进攻
├─ 敌人从边缘平台进攻
├─ 逐个平台争夺
└─ 雇佣兵自动战斗

阶段3: 战斗
├─ Combat雇佣兵输出伤害
├─ Support修复受损平台
├─ Medical治疗受伤队友
└─ Intel标记弱点

阶段4: 结果
├─ 成功: 保留战利品，平台受损但可修复
├─ 失败: 失去资源，雇佣兵可能死亡
└─ 重建: 修复平台，重新部署
```

### 策略深度

**平台布局策略**:
```
外层防线: Combat平台 (火力)
    ↓
中层防线: Support平台 (补给)
    ↓
内层防线: Medical平台 (治疗)
    ↓
核心: HQ (必须守住)
```

**雇佣兵部署策略**:
```
前线防御: Combat雇佣兵 + Combat平台
后方支援: Medical雇佣兵 + Medical平台
侧翼支援: Intel雇佣兵 + Intel平台
机动支援: Support雇佣兵 + Support平台
```

## 🌊 Roguelike深海远征

### 远征队组织

**队伍配置** (3-5雇佣兵):
```
标准配置:
├─ 2 Combat (主力输出)
├─ 1 Support (修复/补给)
├─ 1 Medical (治疗)
└─ 1 Intel (情报/预警)

高风险配置:
├─ 3 Combat (极限输出)
└─ 2 Medical (保命)
```

### 5层深海结构

**Layer 1: 浅海 (新手教程层)**
```
难度: ★☆☆☆☆
敌人: 弱海怪
奖励: 普通资源
特殊: 教学关卡

战斗:
├─ 3个小型平台战场
├─ 1个Boss战 (弱海怪)
└─ 可随时撤退
```

**Layer 2: 中海 (标准难度)**
```
难度: ★★☆☆☆
敌人: 中等海怪 + 海盗
奖励: 稀有资源
特殊: 随机事件

战斗:
├─ 5个平台战场
├─ 2个Boss战
├─ 随机宝箱
└─ 需要策略决策
```

**Layer 3: 深海 (困难)**
```
难度: ★★★☆☆
敌人: 强海怪 + 古代遗迹守卫
奖励: 史诗资源 + 稀有雇佣兵
特殊: 独特机制

战斗:
├─ 7个复杂平台战场
├─ 3个Boss战
├─ 特殊地形效果
└─ 伤亡率高
```

**Layer 4: 深渊 (极难)**
```
难度: ★★★★☆
敌人: Boss级海怪 + 环境伤害
奖励: 传说资源 + 特殊能力
特殊: 高风险高回报

战斗:
├─ 10个高强度战场
├─ 连续Boss战
├─ 环境危害 (水压/暗流)
└─ 死亡率极高
```

**Layer 5: 海底墓场 (秘密)**
```
难度: ★★★★★
敌人: 古神级Boss
奖励: 独特模块 + 终极能力
特殊: 一生只能挑战一次

战斗:
├─ 超长Boss战 (10分钟+)
├─ 多阶段战斗
├─ 全员牺牲风险
└─ 传说级奖励
```

### 远征战斗机制

**与基地防御的区别**:
```
基地防御:
├─ 使用你建造的平台
├─ 平台提供buff
└─ 防御战

深海远征:
├─ 使用随机生成的平台
├─ 平台无buff (纯雇佣兵能力)
└─ 进攻战
```

**战斗过程**:
```
遭遇敌人:
├─ 显示敌人信息 (HP/攻击力/特殊能力)
├─ 显示你的战力 (雇佣兵配置)
└─ 选择: 战斗/规避/撤退

自动战斗:
├─ Combat雇佣兵自动攻击
├─ Medical自动治疗
├─ Support自动修复装备
└─ Intel自动预警

战斗结果:
├─ 完胜: 无伤亡, 150%奖励
├─ 胜利: 轻伤, 100%奖励
├─ 惨胜: 重伤, 50%奖励
└─ 失败: 雇佣兵死亡, 0奖励
```

## 🎯 为什么这样设计好玩?

### 1. 基地即战场 (创新!)

**独特性**:
- ✅ 每个玩家的基地 = 独特的战斗地图
- ✅ 平台布局 = 防御策略
- ✅ 不是预设地图，是玩家自己创造的

**策略深度**:
```
我的基地设计:
├─ 外围全Combat平台 (火力网)
├─ 中间一圈Support (补给线)
├─ 核心Medical + Intel (支援中心)
└─ HQ在最深处 (必须守住)

你的基地设计:
├─ 分散式布局 (降低风险)
├─ 每个分支自给自足
└─ 灵活防御
```

### 2. 无装备系统 (简化!)

**为什么不需要装备?**
- ✅ 平台本身就是"装备"
- ✅ 雇佣兵专业 = 战斗风格
- ✅ 技能等级 = 力量

**对比传统RPG**:
```
传统RPG:
├─ 获得装备
├─ 强化装备
├─ 更换装备
├─ 装备搭配
└─ 复杂度高

本游戏:
├─ 建造平台 (一次性)
├─ 分配雇佣兵 (策略)
├─ 平台协同 (组合)
└─ 简单但深度
```

### 3. 双系统互补

**基地建造**:
```
长期目标:
├─ 扩展基地
├─ 优化布局
├─ 招募雇佣兵
└─ 提升战力
```

**深海远征**:
```
短期刺激:
├─ 随机战斗
├─ 即时奖励
├─ 死亡风险
└─ 每次不同
```

**协同效应**:
```
基地 → 提供雇佣兵和资源
    ↓
远征 → 获得新平台图纸和资源
    ↓
基地 → 建造更强的平台
    ↓
远征 → 挑战更深的层次
```

### 4. 重玩价值

**每次都不同**:
- ✅ 基地布局可以重新设计
- ✅ 雇佣兵随机招募
- ✅ 远征随机生成
- ✅ 专业能力组合策略

**长期目标**:
- ✅ 解锁所有Layer
- ✅ 完美基地布局
- ✅ 收集所有特殊能力
- ✅ 挑战海底墓场

## 🎮 具体玩法示例

### 一次完整的基地防御战

```
阶段1: 预警 (Intel平台触发)
├─ "海盗舰队正在接近！预计30秒后到达！"
├─ 显示进攻路线 (从东侧Combat平台)
└─ 玩家快速部署雇佣兵

阶段2: 部署
├─ 东侧Combat平台: 部署2个Combat雇佣兵
├─ 相邻Support平台: 部署1个Support雇佣兵
├─ 后方Medical平台: 部署1个Medical雇佣兵
└─ Intel平台: 1个Intel雇佣兵 (预警)

阶段3: 战斗开始
├─ 海盗船靠近东侧平台
├─ Combat雇佣兵自动开火
├─ 支持平台炮塔协同射击
└─ 第一波海盗被击退

阶段4: 战斗升级
├─ 第二波海盗更强
├─ 东侧Combat平台受损
├─ Support雇佣兵修复平台
├─ Medical雇佣兵治疗受伤的Combat
└─ Intel雇佣兵预警: "Boss海盗船接近！"

阶段5: Boss战
├─ 海盗船长出现 (Boss级)
├─ 所有雇佣兵全力攻击
├─ Medical雇佣兵拼命治疗
├─ Support雇佣兵紧急修复
└─ 最后一刻, Combat雇佣兵暴击! 胜利!

阶段6: 战后
├─ 获得大量战利品 (资源 + GMP)
├─ 雇佣兵获得经验
├─ 平台受损 (需要用资源修复)
└─ 等待下一波攻击
```

### 一次深海远征

```
准备阶段:
├─ 从基地选择5个雇佣兵:
│   ├─ 2 Combat (5星, "致命一击")
│   ├─ 1 Support (4星, "紧急增援")
│   ├─ 1 Medical (3星, "群体治疗")
│   └─ 1 Intel (5星, "战术分析")
├─ 消耗100 Materials + 50 Fuel (补给)
└─ 出发!

Layer 1: 浅海
├─ 遭遇3个弱海怪
├─ Combat雇佣兵轻松击败
├─ 发现宝箱 (获得材料)
└─ 决策: 继续前进? 是

Layer 2: 中海
├─ 遭遇海盗船 (中等难度)
├─ Combat雇佣兵受伤
├─ Medical治疗
├─ Support修复装备
└─ Boss战: 中等海怪
└─ 艰难胜利, Medical重伤

决策点: 继续还是撤退?
└─ 选择: 撤退 (保留50%战利品)
└─ 返回基地

返回基地:
├─ 携带回资源
├─ Medical雇佣兵需要休息恢复
├─ Combat雇佣兵获得经验
└─ 解锁Layer 3新信息
```

## 🎯 总结

### 核心概念

**"你建设的海上基地，就是你的战场，你的堡垒，你的家"**

- 基地不是静态的背景，是动态的战斗地图
- 雇佣兵不是数据，是你在战斗中的伙伴
- 远征不是附送的，是核心的双系统之一

### 为什么好玩?

1. ✅ **创造力**: 每个玩家建造不同的基地
2. ✅ **认同感**: 基地是你创造的，防守它有情感投入
3. ✅ **策略性**: 布局 + 雇佣兵组合 = 无限可能
4. ✅ **刺激感**: Roguelike的随机性和死亡风险
5. ✅ **成长感**: 基地越来越大，越来越强

### 与其他游戏的区别

**vs 传统基地建设**:
- 传统: 基地只是资源生产
- 本游戏: 基地是战斗地图

**vs 传统Roguelike**:
- 传统: 角色装备驱动
- 本游戏: 平台布局驱动

**vs 传统RPG**:
- 传统: 装备系统复杂
- 本游戏: 无装备，靠策略
