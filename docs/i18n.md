# Internationalization (i18n) Guide

This project uses a centralized text system for easy localization.

## How to Use

### Getting Text

```gdscript
# Simple text retrieval
var text = TextData.get("ui_materials", [100])  # Returns: "Materials: 100"

# Text without parameters
var raw_text = TextData.get_raw("ui_combos")  # Returns: "Combos: %d"

# Platform type names
var platform_name = TextData.platform_type_name("R&D")  # Returns: "R&D"

# Difficulty names
var difficulty = TextData.difficulty_name("easy")  # Returns: "Easy"
```

### Text Key Format

Text keys use the format: `category_keyname`

**Categories:**
- `ui_*` - User interface text
- `platform_*` - Platform type names
- `msg_*` - System messages
- `expedition_*` - Expedition mission names
- `difficulty_*` - Difficulty names

### Adding New Text

1. Open `scripts/text_data.gd`
2. Add your text to the appropriate dictionary:
```gdscript
const UI = {
    my_new_text = "My New Text: %d"
}
```
3. Use it in code:
```gdscript
TextData.get("ui_my_new_text", [42])
```

## Future Localization

To add support for other languages:

1. Create language-specific dictionaries (e.g., `UI_ZH_CN`, `UI_ES`)
2. Add a language selector to `TextData`
3. Modify `get()` to return text based on current language

Example:
```gdscript
static func get(key: String, args: Array = []) -> String:
    var lang = OS.get_locale_language()  # e.g., "zh", "en", "es"
    var text = _get_text_from_key(key, lang)  # Add lang parameter
    if args.size() > 0:
        text = text % args
    return text
```

## Benefits

- **Centralized**: All text in one place
- **Type-safe**: Compile-time checking
- **Easy to update**: Change text without hunting through code
- **Translation-ready**: Simple structure for future language support
- **Formatting support**: Built-in parameter substitution

## Examples

### Before (Hardcoded)
```gdscript
label.text = "Materials: %d" % materials
print("Base has reached maximum platform count (%d)" % max)
```

### After (Localized)
```gdscript
label.text = TextData.get("ui_materials", [materials])
print(TextData.get("msg_build_failed_base_full", [max]))
```
